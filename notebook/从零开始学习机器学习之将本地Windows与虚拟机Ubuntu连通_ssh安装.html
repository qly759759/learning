2.安装

3.测试

二、Linux（Ubuntu）下的ssh安装

1.下载—命令行方式

sudo apt-get install openssh-server

2.测试

  • 1.先输入ifconfig 查看ubuntu的IP地址

  • 2.在windows cmd 中 ping 该ip地址

  • 3.输入 ssh 用户@地址
