一、函数

1.作用

  • 用来将一段功能代码封装到一起，并给他起个名字，在使用时，只需要使用代码的名字即可。

2. 好处

  • 提高代码的开发效率
  • 使代码更加简洁
  • 提高代码的复用率

3.定义和调用

  • 定义

     def 函数名（参数列表）:
 (缩进)函数功能代码…
 	   ……

  • 调用

     	 函数名（）

4. 调用过程

		1)函数在定义时，函数体中的代码时不执行 的
		2)当产生函数调用时，程序才会跳转到函数体中执行函数的代码
		3)函数体中的代码执行完成后，回到调用处

5.函数说明文档

def show():
    '''
    show 函数的作用是用来显示一个字符串
    '''

    print(1)


show()

help(show)

#按住crtl建点击函数名即可查看函数说明文档，也可以通过help方法查看

6.实参和形参

		实参：实际参数，是在函数调用时，括号中出现的数据
		形参：函数被调用时，接收函数调用传递过来的实参数据
		实参和形参在传递时要一一对应

7.函数返回值练习

def get_num():
    a = input('请输入一个数字：')
    return int(a)# input函数得到的是字符串类型，必须转换

8.传参练习

def sum_num(a, b):
    return int(a)+int(b)


print('请输入要计算总和的两个数字：')
a = input()
b = input()
sum_num(a, b)
print(sum_num(a, b))
